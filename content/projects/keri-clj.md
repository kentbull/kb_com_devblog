A Clojure implementation of the Key Event Receipt Infrastructure (KERI) protocol. Brings KERI's decentralized key management infrastructure to the JVM ecosystem with idiomatic Clojure APIs. Enables building secure, end-verifiable identity systems using functional programming paradigms and Clojure's powerful data-oriented approach.
